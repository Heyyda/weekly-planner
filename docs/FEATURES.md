# Фичи — Личный Еженедельник

## MVP (v0.1.0)
Минимальный работающий продукт — локальная версия без сервера.

- [ ] Окно-sidebar: выезжает справа, анимация slide-in/out
- [ ] Полоска-триггер (6px) видна когда панель скрыта
- [ ] 5 рабочих дней (Пн-Пт), сворачиваемые секции
- [ ] Сегодняшний день развёрнут и подсвечен по умолчанию
- [ ] Добавление задач (Enter для быстрого добавления)
- [ ] Checkbox — выполнение задачи, зачёркивание текста
- [ ] Навигация по неделям (стрелки ◀ ▶)
- [ ] Заголовок: "Неделя N (дата-дата месяц)"
- [ ] Тёмная тема по умолчанию
- [ ] Локальное хранение в JSON (%APPDATA%)
- [ ] System tray иконка
