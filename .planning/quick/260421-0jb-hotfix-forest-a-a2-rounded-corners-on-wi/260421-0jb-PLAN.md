    2. Добавить новый метод:
    ```python
    def _apply_window_region_rounded(self) -> None:
        """Win7+: GDI SetWindowRgn для rounded corners.

        Работает на Win10/Win11 одинаково. Silent fail — окно останется прямоугольным.
        SetWindowRgn takes ownership of HRGN — DeleteObject вручную НЕ нужен.
        """
        try:
            hwnd = ctypes.windll.user32.GetParent(self._window.winfo_id())
            w = self._window.winfo_width()
            h = self._window.winfo_height()
            if w <= 1 or h <= 1:
                # Окно ещё не получило размеры — повторим после idle
                self._window.after_idle(self._apply_window_region_rounded)
                return
            r = self.WINDOW_CORNER_RADIUS
            # CreateRoundRectRgn(x1, y1, x2, y2, cornerWidth, cornerHeight)
            # x2/y2 — exclusive, потому +1
            rgn = ctypes.windll.gdi32.CreateRoundRectRgn(0, 0, w + 1, h + 1, r, r)
            # SetWindowRgn(hwnd, hrgn, bRedraw) — True = перерисовать
            ctypes.windll.user32.SetWindowRgn(hwnd, rgn, True)
            logger.debug("MainWindow rounded corners applied via SetWindowRgn (r=%d)", r)
        except Exception as exc:
            logger.debug("SetWindowRgn failed (non-Windows?): %s", exc)
    ```

    3. В `_init_frameless_style`: **заменить** `self._window.after(self.DWM_CORNER_DELAY_MS, self._apply_dwm_rounded_corners)` на вызов GDI-варианта:

    Старый `_apply_dwm_rounded_corners` **можно удалить** (он мёртвый код на Win10 и на Win11 SetWindowRgn тоже работает). Если хочешь подстраховки — оставь, но вызывай только GDI.

    ```python
    # Добавить __init__: self._rgn_reapply_job: Optional[str] = None

        if self._rgn_reapply_job is not None:
            try:
                self._window.after_cancel(self._rgn_reapply_job)
    ```
