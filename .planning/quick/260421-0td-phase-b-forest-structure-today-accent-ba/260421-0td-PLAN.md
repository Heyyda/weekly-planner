   Семантика: в forest_light `bg_primary` = #EEE9DC (cream) — высокий контраст на

   ключа в палитре нет — `bg_primary` семантически эквивалентен и не требует расширять themes.py.

2. **Hover на 🗑 → accent_overdue:**
   В `_icon_hover(btn, entering)` сейчас единая логика:
   ```python
   color = self._theme.get("accent_brand") if entering else (...)
   ```
   Нужно разделить поведение по `btn is self._del_btn`:
   ```python
   def _icon_hover(self, btn: ctk.CTkLabel, entering: bool) -> None:
       if btn is None or not btn.winfo_exists():
           return
       try:
           if entering:
               hover_color = (
                   self._theme.get("accent_overdue")
                   if btn is self._del_btn
                   else self._theme.get("accent_brand")
               )
               color = hover_color
           else:
               color = self._theme.get("text_secondary") if self._hover else self._theme.get("text_tertiary")
           btn.configure(text_color=color)
       except tk.TclError:
           pass
   ```
