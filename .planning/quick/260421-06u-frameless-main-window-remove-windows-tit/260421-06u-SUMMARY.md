# Phase 260421-06u Plan 01: Frameless Main Window Summary

Один-строчное описание: MainWindow потерял нативный Windows title bar и получил кастомную 28px шапку в стиле Forest (3px accent-полоса, "Еженедельник", ✕) с drag-логикой, persist-позицией, hover-эффектом на ✕ и DWM rounded corners для Win11 (silent fail на Win10).

## Что сделано

Модифицирован только `client/ui/main_window.py` — 185 строк добавлено:

### Новые константы класса `MainWindow`
- `TITLE_BAR_HEIGHT = 28`
- `ACCENT_STRIP_WIDTH = 3`
- `FRAMELESS_INIT_DELAY_MS = 100` — Pitfall 1 (Win11 DWM timing), копия идиомы из `OverlayManager.INIT_DELAY_MS`
- `DWM_CORNER_DELAY_MS = 150` — дополнительная задержка перед `DwmSetWindowAttribute` чтобы HWND полностью инициализировался
- `DWMWA_WINDOW_CORNER_PREFERENCE = 33`
- `DWMWCP_ROUND = 2`

### Новые поля инстанса (инициализируются в `__init__` перед `_day_sections`)
- `_title_drag_offset_x`, `_title_drag_offset_y` — drag-оффсет курсора
- `_title_bar`, `_title_accent_strip`, `_title_label`, `_title_close_btn`, `_title_separator` — ссылки на виджеты шапки (для перекрашивания в `_apply_theme`)

### Новые методы (6 шт. — все из плана done-criteria)
1. **`_init_frameless_style(self)`** — вызывается через `after(100ms, ...)` в конце `__init__`; делает `overrideredirect(True)`, переприменяет topmost, планирует `_apply_dwm_rounded_corners` ещё через 150ms.
2. **`_apply_dwm_rounded_corners(self)`** — `GetParent(winfo_id()) → DwmSetWindowAttribute` обёрнут в `try/except` с `logger.debug` (silent fail на Win10 и не-Windows).
3. **`_build_title_bar(self, parent)`** — строит 28px шапку: 3px accent strip (`accent_brand`), label "Еженедельник" (`text_secondary`), ✕ кнопка 28x28 (`text_tertiary` → `accent_overdue` на hover с `bg_secondary` фоном), 1px separator (`bg_tertiary`).
4. **`_on_title_drag_start(event)`** — запоминает offset курсора до левого-верхнего угла окна.
5. **`_on_title_drag_motion(event)`** — `self._window.geometry(f"+{x}+{y}")`; без clamp к virtual desktop (для большого окна clamp не нужен).
6. **`_on_title_drag_end(event)`** — вызывает `self._save_window_state()` — тот же путь что и WM_DELETE_WINDOW.

### Изменения в существующих методах
- **`__init__`**: добавлен вызов `self._window.after(self.FRAMELESS_INIT_DELAY_MS, self._init_frameless_style)` после Ctrl+Space binding.
- **`_build_ui`**: добавлен `self._build_title_bar(self._root_frame)` между `self._root_frame.pack(...)` и `self._week_nav = WeekNavigation(...)`.
- **`_apply_theme`**: расширен (НЕ заменён) — после существующего блока перекрашивания окна/root_frame добавлен блок перекрашивания всех виджетов шапки через внутренний helper `_col(key)` с fallback к `self._theme.get(key)` (потому что `__init__` передаёт palette с 4 ключами, а шапке нужно 5 — `text_secondary`, `text_tertiary`, `bg_tertiary` дополнительно).

### Биндинги
- `<Button-1>` на ✕ → `self._on_close()` (то же что `WM_DELETE_WINDOW` = hide в трей)
- `<Enter>` / `<Leave>` на ✕ → переключение между idle / hover цветами
- `<ButtonPress-1>` / `<B1-Motion>` / `<ButtonRelease-1>` на `_title_bar`, `_title_label`, `_title_accent_strip` — drag (НЕ на `_title_close_btn` — чтобы клик по ✕ не трактовался как drag-start)

## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 1 - Bug] Defensive fallback в `_apply_theme` для частичного palette**
- **Found during:** Task 1 (прогон test suite после первой имплементации)
- **Issue:** `__init__` передаёт в `_apply_theme` dict только с 4 ключами (`bg_primary`, `text_primary`, `bg_secondary`, `accent_brand`). Моя первая реализация использовала `palette.get("text_secondary")` → возвращает `None` → `CTkLabel.configure(text_color=None)` бросает `ValueError("color is None, for transparency set color='transparent'")`.
- **Fix:** Добавлен внутренний helper `_col(key)` в `_apply_theme`, который fallback'ается к `self._theme.get(key)` если ключа нет в переданном palette dict.
- **Files modified:** `client/ui/main_window.py` (только расширение уже изменённого метода)
- **Verification:** `python -m pytest client/tests/ui/test_main_window.py` — все 23 теста проходят.

Других отклонений нет — план выполнен точно как написан.
